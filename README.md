# COUP Neural AI Submission

This folder contains only the files that I created or wrote for my COUP project work. I separated these files from the original `mycoup` project folder because the rest of the project contains starter code, base game files, images, rules, and other materials written or provided by my professor.

## Files Included

- `neural-ai.js` - my neural-network-based AI implementation for COUP.
- `neural-ai.txt` - my notes and explanation for the neural AI work.
- `experiment-guide.txt` - my experiment guide and process notes.
- `COUP_Project_Report.pdf` - my project report.
- `coup_neural_ai_weights_v3_64_32 (1).json` - neural AI weights file.
- `coup_neural_ai_weights_v3_64_32after100000.json` - neural AI weights after additional training.
- `coup_neural_ai_weights_v3_32_16.json` - neural AI weights file.
- `coup_neural_ai_weights_v3_32_16after100000.json` - neural AI weights after additional training.

## What I Did

For this project, I worked on adding and documenting a neural-network-based AI player for the COUP game. My work focused on the neural AI implementation, training/experiment notes, saved model weights, and the final project report explaining the approach and results.

The included weight files represent trained versions of the neural AI using different network sizes and training stages. These files are included because they are part of the AI behavior I developed and tested.

## Files Not Included

I did not include the other files from the original `mycoup` folder because those files were written or provided by my professor. They include the base COUP game implementation, existing AI files, images, rules, HTML files, and other starter project materials.

This submission folder is intended to show only my own contribution while leaving the professor-provided code and assets outside of the submitted work.
